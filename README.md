# kyofi
File manager with ssh, s3, ftp, ssh+docker support.
## Main goal 
To provide identical workflow while working with different protocols.

## Deafult keybindings
- `j`/`down` - move down
- `k`/`up` - move up
- `ctrl+l`/`right` - move focus right
- `ctrl+h`/`left` - move focus left
- `y` - copy
- `p` - paste
- `d` - delete
- `ctrl+n` - protocol selection
